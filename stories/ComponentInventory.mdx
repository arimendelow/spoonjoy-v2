## Quick Reference

Need something fast? Here's the TL;DR:

| I need to... | Use this |
|-------------|----------|
| Collect text input | [Input](/docs/ui-input--docs) or [Textarea](/docs/ui-textarea--docs) |
| Let users pick from options | [Select](/docs/ui-select--docs), [Listbox](/docs/ui-listbox--docs), or [Combobox](/docs/ui-combobox--docs) |
| Toggle something on/off | [Switch](/docs/ui-switch--docs) or [Checkbox](/docs/ui-checkbox--docs) |
| Show a confirmation | [Alert](/docs/ui-alert--docs) |
| Show a form in an overlay | [Dialog](/docs/ui-dialog--docs) |
| Display tabular data | [Table](/docs/ui-table--docs) |
| Show key-value details | [DescriptionList](/docs/ui-descriptionlist--docs) |
| Display validation errors | [ValidationError](/docs/ui-validationerror--docs) |
| Create a page layout | [StackedLayout](/docs/ui-stackedlayout--docs) or [SidebarLayout](/docs/ui-sidebarlayout--docs) |
| Add navigation | [Navbar](/docs/ui-navbar--docs) or [Sidebar](/docs/ui-sidebar--docs) |

## Contributing

Want to add a component to this inventory? Here's the recipe:

1. Create your component in `app/components/`
2. Write stories in `stories/YourComponent.stories.tsx`
3. Add comprehensive JSDoc documentation (with personality, please)
4. Update this inventory
5. Get a code review from someone who's had coffee

Remember: A component without stories is like a recipe without tasting. Sure, it might work, but you'll never know until someone tries it.
